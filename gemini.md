# Ivy Gomez Skin Specialist Website

This project is a website for a skin specialist named Ivy Gomez. It is built with React and TypeScript.

## Project Goals

*   Build a professional and modern website to showcase Ivy Gomez's services.
*   Implement features such as online booking, a gallery of work, and detailed service descriptions.
*   Ensure the website is responsive and works well on all devices.
